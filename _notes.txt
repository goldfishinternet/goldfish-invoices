Notes
**********************************
Laravel prototype is based on a QuickAdminPanel gernerated code Laravel 9 which requires PHP 8+
It has a standard breeze type architecture and I have been working on a docker/sail dev environment.

Uses Stripe/Cashier for subscriptions

Sail
*********************************
alias sail='[ -f sail ] && bash sail || bash vendor/bin/sail'
sail up
sail down
sail artisan migrate:fresh --seed

*********************************
